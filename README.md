# ELS-Hummer-Pack
FiveM/GTA V Mod 
			
How to add to Fivem 

(Addon)Fivem only

add this to your server.cfg

start stewh1pack

Drop the ELS files under vcf

	"h1s1.xml",
	"h1s2.xml",
	"h1s3.xml",
	"h1s4.xml",
	"h1p1.xml",
	"h1p2.xml",
	"h1p3.xml",
	"h1p4.xml",
